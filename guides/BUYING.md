# 买入代币指南

创建代币后，有几种方法可以买入：

## 方法一：使用脚本（推荐）

```bash
npx tsx scripts/buy-token.ts <代币地址> <BNB数量>
```

**示例：**
```bash
npx tsx scripts/buy-token.ts 0x1234567890abcdef... 1
```

**参数：**
- `代币地址` - 刚创建的代币合约地址
- `BNB数量` - 要花费的 BNB 数量（如 1 表示 1 BNB）

**如何获取代币地址：**
1. 查看创建交易的 receipt
2. 或在 BSCScan 上查看交易日志
3. 事件 `TokenCreated` 中的地址

## 方法二：FOUR.MEME 前端（最简单）

1. 打开 https://four.meme
2. 连接钱包
3. 搜索你的代币（按名称或地址）
4. 点击 "Buy"
5. 输入金额，确认交易

## 方法三：手动合约调用

如果你熟悉合约交互，可以直接调用：

```solidity
Factory.buyToken(
  address token,    // 代币地址
  uint256 minAmount // 最小接收数量（0 表示接受任何数量）
)
```

** payable **，发送 BNB 作为购买资金

## 注意事项

1. **创建后等待确认**
   - 代币创建交易需要确认后才能买入
   - 等待 5-10 秒后再尝试买入

2. **滑点设置**
   - 脚本中设置 minTokens = 0（接受任何数量）
   - 如需设置滑点，修改脚本中的 minTokens 参数

3. **Gas 费用**
   - 买入需要额外的 BNB 作为 gas
   - 建议预留 0.01-0.02 BNB

4. **买入限制**
   - 新代币可能有买入限制（前 10 分钟）
   - 查看 FOUR.MEME 页面上的具体规则

## 常见问题

### Q: 买入失败，显示 revert
**原因：**
- 代币创建交易还未确认
- 代币已经毕业（迁移到 PancakeSwap）
- 达到买入上限

**解决：**
- 等待创建交易确认
- 在 PancakeSwap 上交易
- 检查买入限制

### Q: 不知道代币地址
**获取方法：**
1. 在 BSCScan 搜索你的钱包地址
2. 找到创建交易
3. 查看事件日志中的 TokenCreated 事件

### Q: 买入数量太少
**原因：**
- 价格滑点
- 交易税费
-  bonding curve 机制

**解决：**
- 增加买入金额
- 调整滑点容忍度
- 分多次买入
